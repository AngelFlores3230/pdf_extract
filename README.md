# pdf_extract
Repositorio experimental para la comparación de librerías Python de extracción de texto desde documentos PDF digitales. Incluye implementaciones reproducibles de PyMuPDF, pdfminer.six, pdfplumber y pypdf, notebooks en Google Colab, métricas de evaluación,   y documentos PDF de prueba de 10, 25, 50 y 100 páginas utilizados en el estudio.
